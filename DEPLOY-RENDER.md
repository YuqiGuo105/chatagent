One Blueprint click deploys **both** services from this repo. Total cost: **$0/month**.

## Prerequisites

- GitHub account (you have `YuqiGuo105/chatagent`)
- Supabase project (you have `iyvhmpdfrnznxgyvvkvx`, region `us-west-1`)

## 1) Push `render.yaml` to main

```bash
git add render.yaml Dockerfile chatagent-app/src/main/resources/application.yaml writer-service .github/workflows/writer-service-ci.yml
git commit -m "feat: render Blueprint for chatagent + writer-service"
git push origin main
```

## 2) Create Blueprint in Render

1. Render dashboard ‚Üí **New** ‚Üí **Blueprint**
2. Connect this repo (`YuqiGuo105/chatagent`)
3. Render reads `render.yaml`, shows 2 services (chatagent, writer-service)
4. Click **Apply** ‚Üí Render starts building both

First build takes ~5‚Äì10 min (Docker layer cache empty).

## Free tier caveats

- **Sleeps after 15 min idle.** First request after sleep takes **30‚Äì60 s** (JVM cold start). Subsequent requests are fast.
- **512 MB RAM.** Both Spring Boot apps fit but are tight. If OOM: bump to Starter ($7/mo).
- **750 hours/month free.** One always-on service uses ~720 h; sleep saves headroom.
- **No persistent disk** ‚Äî fine because state lives in Supabase.

## Avoiding cold starts (optional)

Use a free cron pinger to hit `/actuator/health` every 10 min:

- **GitHub Actions** cron (free): `*/10 * * * *` ‚Üí `curl <url>/actuator/health`
- **UptimeRobot** (free 50 monitors, 5 min interval)
- **cron-job.org** (free)

## When to upgrade

| Symptom | Action |
|---|---|
| Cold starts annoy users | Render Starter for chatagent only ‚Üí $7/mo |
| OOM kills | Render Starter (still 512 MB but no sleep) or Standard 2 GB ‚Üí $25/mo |

| Need multi-region/CDN | Cloudflare Workers in front of Render (free CDN) |

## Troubleshooting

| Symptom | Cause / Fix |
|---|---|
| `password authentication failed for user "postgres"` | Wrong DB password or wrong pooler region. Verify at https://supabase.com/dashboard/project/iyvhmpdfrnznxgyvvkvx/settings/database |
| `ENOTFOUND tenant/user` | Wrong pooler region in `*_DB_URL` |
| `column "payload" is of type jsonb but expression is of type character varying` | Missing `&stringtype=unspecified` in `WRITER_DB_URL` |
| Build OOM | Render free build runner has 4 GB; Maven `-Xmx512m` already set in mvnw. If still OOM, build locally and push image to Docker Hub instead. |
| Health check fails | App boot >5 min; check service logs in Render dashboard |
